Welcome to the Citrine Python client documentation!

This site documents the Python SDK for the Citrine Platform. It provides utilities to upload and manage data and design materials using Sequential Learning. See the getting started guide for a high-level introduction. The workflows section documents how to configure and run artificial intelligence (AI) workflows for materials research and development.

Installation

The latest release can be installed via pip:

pip install citrine

or a specific version can be installed, for example:

pip install citrine==1.0.0
