class citrine.informatics.processors.GridProcessor(name: str, description: str, grid_sizes: Mapping[str, int], session: Optional[citrine._session.Session] = None)¶

Bases: citrine._serialization.serializable.Serializable, citrine.informatics.processors.Processor

Generates samples from the outer product of finite dimensions, then scans over them.

To create a finite set of materials from continuous dimensions, a uniform grid is created between the lower and upper bounds of the descriptor. The number of points along each dimension is specified by grid_sizes.

Parameters

  • name (str) – name of the processor

  • description (str) – description of the processor

  • grid_sizes (dict[str, int]) – the number of points to select along each dimension of the grid, by dimension name
