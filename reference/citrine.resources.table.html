citrine.resources.table module¶

class citrine.resources.table.Table¶

Bases: citrine._rest.resource.Resource

A 2-dimensional projection of data.

Tables are the basic unit used to flatten and manipulate data objects. While data objects can represent complex materials data, the format is NOT conducive to analysis and machine learning. Tables, however, can be used to ‘flatten’ data objects into useful projections.
