citrine.resources.parameter_template module¶

Resources that represent parameter templates.

class citrine.resources.parameter_template.ParameterTemplate(name: str, bounds: gemd.entity.bounds.base_bounds.BaseBounds, uids: Optional[Dict[str, str]] = None, description: Optional[str] = None, tags: Optional[List[str]] = None)¶

Bases: citrine.resources.attribute_templates.AttributeTemplate, citrine._rest.resource.Resource, gemd.entity.template.parameter_template.ParameterTemplate

A parameter template.

Parameters

  • name (str) – The name of the parameter template.

  • bounds (BaseBounds) – Bounds circumscribe the values that are valid according to this parameter template.

  • description (str, optional) – A long-form description of the parameter template.

  • uids (Map[str, str], optional) – A collection of unique IDs.

  • tags (List[str], optional) – Tags are hierarchical strings that store information about an entity. They can be used for filtering and discoverability.
