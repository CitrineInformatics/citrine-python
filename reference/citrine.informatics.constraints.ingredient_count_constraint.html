Bases: citrine._serialization.serializable.Serializable[IngredientCountConstraint], citrine.informatics.constraints.constraint.Constraint

Represents a constraint on the total number of ingredients in a formulation.

Parameters

  • formulation_descriptor (FormulationDescriptor) – descriptor to constrain

  • min (int) – minimum ingredient count

  • max (int) – maximum ingredient count

  • label (Optional[str]) – Optional label to constrain. If specified, then only ingredients with the specified label will count towards the total. Default is None; all ingredients count towards the total
