Bases: citrine.resources.data_concepts.DataConcepts, citrine._serialization.serializable.Serializable, taurus.entity.attribute.property.Property

A property attribute.

Properties are characteristics of a material that could be measured, e.g. chemical composition,

density, yield strength.

Parameters

  • name (str) – Required name of the property. Each property within an object must have a unique name.

  • notes (str) – Optional free-form notes about the property.

  • value (BaseValue) – The value of the property.

  • template (PropertyTemplate) – Property template that defines the allowed bounds of this property. If a template and value are both present then the value must be within the template bounds.

  • origin (str) – The origin of the property. Must be one of “measured”, “predicted”, “summary”, “specified”, “computed”, or “unknown.” Default is “unknown.”

  • file_links (List[FileLink]) – Links to files associated with the property.
