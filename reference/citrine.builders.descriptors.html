static from_templates(project: citrine.resources.project.Project, scope: str)¶

Build a PlatformVocabulary from the templates visible to a project.

All of the templates with the given scope are downloaded and converted into descriptors. The uid values associated with that scope are used as the index into the dictionary. For example, using scope “my_templates” with a template with uids={“my_templates”: “density”} would be indexed into the dictionary as “density”.

Parameters

  • project (Project) – Project on the Citrine Platform to read templates from

  • scope (str) – Unique ID scope from which to pull the template names

Returns

Return type

PlatformVocabulary

citrine.builders.descriptors.template_to_descriptor(template: gemd.entity.template.attribute_template.AttributeTemplate) → citrine.informatics.descriptors.Descriptor¶

Convert a GEMD attribute template into an AI Engine Descriptor.

IntBounds cannot be converted because they have no matching descriptor type. CompositionBounds can only be converted when every component is an element, in which case they are converted to ChemicalFormulaDescriptors.

Parameters

template (AttributeTemplate) – Template to convert into a descriptor

Returns
