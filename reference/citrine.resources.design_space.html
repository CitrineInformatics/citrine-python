validate_write_request(design_space: citrine.informatics.design_spaces.design_space.DesignSpace)¶

Perform write-time validations of the design space registration or update.

EnumeratedDesignSpaces can be pretty big, so we want to return a helpful error message rather than let the POST or PUT call fail because the request body is too big. This validation is performed when the design space is sent to the platform in case a user creates a large intermediate design space but then filters it down before registering it.
